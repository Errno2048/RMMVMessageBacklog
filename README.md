# RMMVMessageBacklog
A simple message backlog plugin for RPG Maker MV.

## Configuration

| Parameter | Default | Description                                                  |
| --------- | ------- | ------------------------------------------------------------ |
| key       | tab     | The key for calling the message backlog window (enabled on the map only) |
